  • httpBlack Jack

    Black Jack game made with Javascript. Cookies are used to save information about the user on the device such as amount of Wins, Losses, Money, etc. Firebase is used to display the names of all players currently on the site and if their name is clicked on, shows their current hand. Features animated card flips and modals. The user's own name is shown in yellow and they can easily change their display name.

    Github | Website

    Here you can see that there is a modal on the screen notifying the user that they are a new visitor. Because they are new, they are given a bank balance of 1000. Uses cookies to determine if they are a new user or not.

    One of the issues with the modal that I have not been able to figure out, is to close the modal. In most cases, you can click the X in the top right, or click any where on the page and it will close. But after having some friends and family, there were two cases of clicking any where on the page would not close the page. Unfortunately, it was on an Android with Chrome, and an iPhone with Safari.

    After the user begins a game, they are greeted with this screen. This allows the player to place a wager using money from their bank. It will check if the amount they are trying to bet is more than what they have and tell them if they don't have enought credits.

    One of the harder things to implement was in the use case of someone placing a bet, say $50, but then changing it to a custom amount. With the original code, it would take out both the custom amount and the preset amount, causing the bank to possibly go negative. Now, it just resets the bet and then applies the custom bet.

    This is what the player will see when playing the game. In the top right is their bank, how much they wagered, and the value of their current hand. The cards animate from the face down card on the right and flip to reveal the value of the card.

    For a while, it was hard to decide how I wanted the cards to go onto the table and how I wanted to animate it. After some research, I found a sleek implementation of a flip card animation and applied it to my use case. Using an offset variable in Javascript, it uses the same animation for all cards but with different position arguments.

    When you win, lose, or wash, this modal will display letting you know if you the outcome of the game. From there, you will be able to start a new game or quit and go back to the main menu.

  • whatshotNote to Key

    A program made in Python that takes the loudest frequency from a given input and maps those frequencies to a keystroke or button press. You can save and load profiles for different environments.

    Github

    This is the screen you will see when you first open the program. The UI is created using Tkinter and has pretty basic functions.
    Detailsmore_vert
    Detailsclose

    There is a lot of small indicators in the UI to let the user know what is currently happening. When you select an input, it has a tick mark next to it. When you press Start, it is highlighted in green and any inputs that get pressed also turn green. The title bar will update with the name of the current mapping profile to give a better indication of the current configuration.
